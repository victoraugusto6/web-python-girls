### Projeto para criar aplicação do site [Django Girls](https://tutorial.djangogirls.org/pt/)

Link: https://webdjangogirls.herokuapp.com/

### <strong>Instruções para instalação</strong>:

#### Criar e ativar ambiente virtual Python (venv):

```python -m venv .venv```

```source .venv/bin/activate```

#### <strong>Instalar dependências</strong>:

```pip install -r requirements.txt```

#### <strong>Instalar dependências, inclusive de desenvolvimento</strong>:

```pip install -r requirements-dev.txt```

#### Copiar variáveis de ambiente:
```cp contrib/env-sample .env```

#### Rodar Django:
```python manage.py runserver```
